
Clario is a web-based paraphrasing and translation application built with **React** and **FastAPI**. It helps users rewrite text with different styles and translate content between languages through a simple and modern interface.

### Frontend
- React
- JavaScript / TypeScript
- CSS
- Vite

### Backend
- FastAPI
- Python
- REST API
